# 2026-05-20
<!-- DAILY_CHECKIN_2026-05-20_START -->
-   LLM (大语言模型)：非确定性的高维“世界模型”，本质上是对人类知识高维压缩的概率机器，擅长语义推理与模式匹配而非精确计算。
    
-   Prompt (提示词)：高维参数空间的方向盘，通过高级语义对模型的概率场进行边界约束与上下文激活，从而拉平认知偏差。
    
-   Workflow (工作流)：确定性的刚性骨架，用代码锁死固定的业务逻辑与任务管线，是扼杀 LLM 幻觉、实现工业级落地的基石。
    
-   Tool Use (工具调用)：大模型的“行动义肢”，通过将自然语言意图转化为结构化 API 参数，让 AI 具备连接外部世界与实时数据的感知能力。
    
-   Agent (智能体)：具备自适应决策的数字生命，将大模型、记忆、规划与工具融为一体，能够自主拆解目标并进行闭环纠错。
    
-   AI Coding (AI 编程)：声明式的开发范式转移，将代码编写降维打击为自然语言的意图声明，使程序员的核心竞争力从“堆砌语法”升维至“架构审美与问题定义”。
