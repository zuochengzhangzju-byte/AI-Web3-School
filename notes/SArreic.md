**AI Basics: Key Terms**

-   **LLM** – Large Language Model (e.g., GPT). A neural network trained on massive text to understand and generate human-like language.
    
-   **Prompt** – The input or instruction you give to an LLM. Good prompts = better answers.
    
-   **Workflow** – A fixed sequence of steps (e.g., prompt → LLM → output). No decisions made by the AI.
    
-   **Agent** – An AI system that can plan, reason, and act autonomously. Agents decide _what to do next_.
    
-   **Tool use** – When an LLM calls external tools (e.g., search, calculator, code runner) to get data or take action.
    
-   **AI coding** – Using AI to write, explain, debug, or refactor code. Example: Cursor, Copilot.
