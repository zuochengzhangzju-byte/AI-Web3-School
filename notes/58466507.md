# 2026-05-26
<!-- DAILY_CHECKIN_2026-05-26_START -->



AI Agent 的记忆依托于和用户的交互，是上下文对话（工作）连贯性的基础。

记忆可以分为：长期有效的偏好+有用的上下文（ChatGPT）、项目的偏好+限制+经验教训（Claude Code），这是基于不同需求的模式

Claude Code 面向程序员，于长执行链路的过程中暴露了memory的不足，存在如下的管理措施（全生命管理，从产生到输出到删除）：

（1）测试维度：Recall 记忆唤起 Revise 记忆处理 Retrieve 记忆提取

（2）构成：时间轴(time axis)、记忆轴（content axis）、正交轴（orthogonal axis）——>Claude表示，在文件memory的意义上，不具备时间轴，尽管这可能是不同的概念

（3）挑战：提取的记忆是否应脱离上下文情景——>记忆越强，越不必回到上下文
