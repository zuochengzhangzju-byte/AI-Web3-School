### **AI x Web3 School 第一期开营仪式学习笔记**

**一、 课程概况与背景**

-   **规模与支持**：本期活动报名人数超过 1050 人。活动由 **ZI** 领先支持，**COMBO** 联合赞助，并得到了 LXDAO、Star、Panda 等多个项目方及社区的支持。
    
-   **核心愿景**：探索 AI 与 Web3 的交叉领域，包括 AI Agent 之间的支付结算、隐私保护、链上动作执行以及开发者工作流的重写。
    
-   **活动阶段**：
    
    -   **Bootcamp (5月15日-31日)**：为期两周的密集学习期。
        
    -   **Hackathon**：第三周进入黑客松/共建阶段。
        
    -   **Demo Day**：整个项目的终点，展示学习成果与作品。
        

**二、 学习机制与打卡要求**

-   **学习平台**：学员需在专属学习面板查看任务、日历和 RO 链接。建议订阅 Google 日历以防错过活动。
    
-   **打卡规则**（5月18日正式开始）：
    
    -   **出勤要求**：每周至少打卡 5 天，允许请假 2 天。**若连续 3 天未打卡将被移出积分排名系统**。
        
    -   **内容质量**：打卡内容需有质量，拒绝“已打卡”等敷衍文字，有效笔记将沉淀到个人 Profile 中。
        
-   **积分系统**：任务分为通用任务、AI 方向、Web3 方向及综合任务。**高积分不仅是学习证明，还将解锁“隐藏福利”与惊喜**。
    

**三、 核心学习资源与工具**

-   **Hermes Agent (学习助手)**：
    
    -   学员可以配置自己的 Hermes Agent 作为贴身管家。
        
    -   **功能**：同步每日课程任务、安排学习计划、记录学习日志并自动提交到个人的 GitHub 仓库。
        
-   **学习地图 (Roadmap)**：
    
    -   团队“手搓”出的路线图，涵盖 AI 基础（大模型、Prompt）、Web3 基础及 AI x Web3 的交叉应用方向。
        
    -   提供具体的 Demo 方向参考，如 Commerce、Security、Governance 等。
        

**四、 专家与助教支持**

-   **Co-learning (Coring) 时间**：每周一、三、五（或六）晚上 **7:00 - 8:00 (UTC+8)**。
    
    -   **注意**：Coring 环节侧重答疑，**没有录播**，建议准时参加以实时解决技术难题。
        
-   **助教团队**：包括 Swing、马铃薯、Jack、小海等专家，覆盖 AI 原生系统开发、Web3 协议研究等领域。
    

**五、 重磅技术分享：ZI 与 GLM 模型**

-   **技术沿革**：ZI 团队分享了 GLM (General Language Model) 的起源，其 GLM 4.5 是首个融合了识图、推理和代码（ARC）能力的开放权重模型。
    
-   **GLM 5.1 演示**：
    
    -   展示了 **GLM 5.1 在 coding 领域的强大能力**，在测试中稳居前列。
        
    -   **自主性 demo**：模型根据单一指令，在无人工干预的情况下，历时 8 小时自主构建了一个包含 50 多个应用程序的功能性 Linux 桌面。
