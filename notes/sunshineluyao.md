AI × Web3 School 打卡

主题：Long-term Memory for AI Agents

今天参加了关于 AI Agent 长期记忆的分享，核心收获是：没有记忆的 Agent 只能做一次性任务，有长期记忆的 Agent 才能形成稳定的"数字身份"，这对 Web3 场景特别重要，比如 DAO 治理中 Agent 需要记住自己的投票立场和历史偏好。

对我 Week 2 的启发：之前我只关注 Agent 的实时能力，现在意识到长期记忆层可能才是 Web3 Agent 的差异化关键。链上数据天然持久，但如何让 Agent 理解并调用这些历史信息，是个值得深挖的方向。

下一步：在问题地图里加入"记忆层"维度，看看能不能和 Hackathon 选题结合。
